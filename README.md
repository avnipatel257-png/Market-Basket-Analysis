# Market-Basket-Analysis

Introduction 
Market Basket Analysis (MBA) is a data mining technique used to identify relationships and patterns between products that are frequently purchased together by customers. It plays a crucial role in the retail and e-commerce industries by helping businesses understand customer buying behavior and improve sales strategies through product recommendations and cross-selling techniques. 
With the rapid growth of data in modern businesses, analyzing transaction data has become essential for making informed decisions. Market Basket Analysis uses association rule mining algorithms such as the Apriori algorithm to discover meaningful relationships between items. These relationships are measured using key metrics like Support, Confidence, and Lift, which indicate the frequency, probability, and strength of association between products. 
In this project, Market Basket Analysis is performed using transactional data to identify strong product associations. The Apriori algorithm is implemented (using Python) to generate association rules, and the results are visualized using Microsoft Power BI. The interactive dashboard includes various visualizations such as scatter plots, bar charts, doughnut charts, and key influencer visuals, allowing users to explore insights effectively. 
Additionally, advanced features like drill-through functionality are used to provide detailed product-level analysis, making the dashboard more interactive and userfriendly.  
. Objectives 
The main objectives of this project are: 
•	To analyze customer transaction data and identify purchasing patterns using Market Basket Analysis. 
•	To apply the Apriori algorithm to generate association rules between products. 
•	To evaluate relationships between items using key metrics such as Support, Confidence, and Lift. 
•	To identify strong product combinations that can be used for cross-selling and recommendation strategies.
